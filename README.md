# 🕒 Reloj Digital Adaptable

Autor: **Christian Lera**

Aplicación de escritorio simple desarrollada en Python con Tkinter que muestra un reloj digital personalizable, con opciones para cambiar entre formato de 12/24 horas y mostrar u ocultar los segundos.

## ✨ Características

- Reloj digital en tiempo real
- Formato de 12 o 24 horas (conmutables)
- Opción para mostrar/ocultar segundos
- Interfaz moderna con tema oscuro
- Ventana redimensionable y centrada
- Fecha completa en español

## 📋 Requisitos

- Python 3.6 o superior
- Tkinter (viene incluido con Python en la mayoría de distribuciones)

## 🚀 Instalación y ejecución

### En Windows (con doble clic):

- **Ejecutar.bat** - Inicia el reloj directamente (abre una ventana cmd)
- **Ejecutar.ps1** - Script PowerShell para iniciar el reloj (más silencioso)

### En cualquier sistema operativo:

```bash
python RelojDigital.py
```

```
Reloj-Digital/
```

## 🎮 Controles

- **Botón "24 Horas / 12 Horas"** - Cambia el formato horario
- **Botón "Ocultar/Mostrar segundos"** - Activa o desactiva la visualización de segundos

## 📝 Notas

- La ventana se ajusta automáticamente cuando cambias entre formatos
- No requiere dependencias externas (solo la biblioteca estándar de Python)
- El reloj se actualiza cada segundo automáticamente
